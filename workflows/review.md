# Review Workflow

Use this when reviewing agent output from Linear.

## Review Questions

- Does the result answer the Linear issue?
- Did the agent stay inside scope?
- Is the mode still correct?
- For explore/prototype, did we learn enough to decide?
- For build, is it production-ready?
- Are tests, screenshots, or demos sufficient?
- Should the work continue, convert, merge, park, or discard?

## Decision Comment

Post a Linear comment with one decision:

- `continue`
- `convert to prototype`
- `convert to build`
- `needs rework`
- `ready to merge`
- `park`
- `discard`
